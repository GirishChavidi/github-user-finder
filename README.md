GitHub User Finder

A modern React application that allows users to search GitHub profiles and explore their repositories using the GitHub API.

The application displays user profile information such as avatar, bio, followers, and public repositories, along with repository details like stars, forks, and language.

Key Features

Search GitHub users instantly

Debounced search for optimized API requests

Display user profile information

View public repositories

Repository pagination

Repository filtering and sorting

Clickable repository links that open on GitHub

Responsive modern UI

Tech Stack

React

JavaScript

GitHub REST API

CSS

Vite

Installation

Clone the repository

git clone https://github.com/GirishChavidi/github-user-finder.git

Go to the project folder

cd github-user-finder

Install dependencies

npm install

Run the application

npm run dev
Environment Setup

Create a .env file in the root directory

VITE_GITHUB_TOKEN=your_github_token_here

This increases the GitHub API rate limit.
